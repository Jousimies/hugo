我关注的 RSS 源

https://sachachua.com/blog/feed/ https://helpdeskheadesk.net/index.xml https://emacstalk.github.io/podcast/index.xml https://kangxiaoning.github.io/index.xml https://karthinks.com/blog/index.xml https://manateelazycat.github.io/feed.xml https://planet.emacslife.com/atom.xml https://emacs-china.org/latest.rss https://www.zhihu.com/rss https://sspai.com/feed 我使用 Emacs 中的 elfeed 阅读一些提供 RSS 订阅的网站，目前关注的主要是 Emacs 相关的的博客。 使用 RSS 的一个原因是现在的信息过载太严重。比如看知乎，有时候一刷就是很长时间，感觉看了很多东西，但是好像没什么作用。听君一席话，如听一席话。看了那么多的废话有什么用啊。保持自己的关注点在自己感兴趣的内容上。 RSS 也有可能信息过载，比如上面的少数派，东西就很杂。再看一段时间，或许就删除了。 ,如《如何阅读一本书》中写的，阅读的目的是什么？接受知讯？还是为了增进自己的理解？只有搞清楚了自己阅读的目的，才知道取舍，有所得有所失。
