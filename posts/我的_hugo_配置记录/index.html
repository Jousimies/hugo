使用什么主题#

选择使用 https://github.com/adityatelange/hugo-PaperMod 这个主题。

git submodule add https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1
echo 'theme = "PaperMod"' >> config.toml

怎么显示 menu bar ？#

调研了大多数的博客，一般显示 Search, Posts, Tags, Categories, Archives 这几个。我的设置如下所示：

在 config.toml 这个配置文件中增加以下几行。

[[menu.main]]
identifier = "Posts"
name = "Posts"
url = "/posts"
weight = 20

[[menu.main]]
identifier = "Search"
name = "Search"
url = "/search"
weight = 10

[[menu.main]]
identifier = "Archives"
name = "Archives"
url = "/archives"
weight = 50

[[menu.main]]
identifier = "Tags"
name = "Tags"
url = "/tags/"
weight = 30

[[menu.main]]
identifier = "Categories"
name = "Categories"
url = "/categories/"
weight = 40

此外，在项目文件夹 content 中增加以下两个文件。

第一个是 search.md

---
title: "Search" # in any language you want
layout: "search" # is necessary
# url: "/archive"
# description: "Description for Search"
summary: "search"
placeholder: "placeholder text in search input box"
---

第二个是 archives.md

---
title: "Archive"
layout: "archives"
url: "/archives/"
summary: archives
---

怎么在首页显示 github, email 等图标#

[params.homeInfoParams]
title = "知易行难"
content = "好记性不如烂笔头"

[[params.socialIcons]]
name = "github"
url = "https://github.com/jousimies"
[[params.socialIcons]]
name = "email"
url = "mailto:duan_n@outlook.com"
[[params.socialIcons]]
name = "RSS"
url = "index.xml"

配置完以上的部分，就可以在首页看到相应的图标。
